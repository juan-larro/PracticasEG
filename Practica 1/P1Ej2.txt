2.b)

<div id="bloque1">Contenido del bloque1</div>
Se ubica dentro del <body>
Elemento: <div>
Etiquetas: Apertura <div> y cierre </div>
Atributo: id(bloque1,no es obligatorio)

2.c)
<img src="" alt="lugar imagen" id="im1" name="im1" width="32" height="32"
longdesc="detalles.htm" />
Se ubica dentro del body
Elemento: img, inserta una imagen
Etiqueta vacía: <img/>
Atributos: src ("", obligatorio) alt (lugar imagen, obligatorio) id (im1, no oblig),
name (im1, no oblig) width (32, no oblig)  height (32, no oblig) longdesc(detalles.html, no oblig)

2.d)
<meta name="keywords" lang="es" content="casa, compra, venta, alquiler " />
<meta http-equiv="expires" content="16-Sep-2019 7:49 PM" />
Se ubica dentro del <head>
Proporciona metadatos
Etiqueta: <meta>
Atributos: 
Primer <meta>: name(keywords,oblig) Lang(es, no oblig), content("casa,compra,venta,alquiler", oblig)
Segundo <meta>: http-equiv(expires, oblig), content("16-Sep-2019 7:49 PM", oblig)

2.e)
<a href="http://www.e-style.com.ar/resumen.html" type="text/html" hreflang="es" charset="utf-8"
rel="help">Resumen HTML </a>
Se ubica en el <body>
Elemento: a, crea un hipervínculo
Etiquetas: Apertura <a> cierre </a>
Atributos: href("http://www.e-style.com.ar/resumen.html", oblig),
type("text/html", no oblig) hreflang(es, no oblig) charset(utf-8, no oblig)
rel("help", no oblig)
